# Paleo

An evolved calculator written in Java.

## Dependencies

You need to have :

* `maven` >= 4.0
* `make`

## Build

To compile the project, just run 

```zsh
$ make
```

In order to generate documentation, run 

```zsh
$ make generate-doc
```

For cleaning, run 

```zsh
$ make clean
```

## Run

For launching the calculator, run 

```zsh
$ make run
``` 

or 

```zsh
$ java -jar paleo-calculator.jar
```
